## Три уровня K8s security

| Уровень | Когда | Что делаем | Инструменты |
|---------|-------|------------|-------------|
| **Pre-deploy** | До деплоя | Сканируем манифесты и кластер на мисконфиги | Kubescape, kube-bench |
| **Admission** | Во время деплоя | Блокируем небезопасные ресурсы | Kyverno |
| **Runtime** | После деплоя | Детектим аномальное поведение | Falco |

Модуль 4.2 покрыл pre-deploy (скан YAML). Этот модуль добавляет **скан живого кластера**, **admission control**, и **runtime detection**.

**Kubernetes Goat** — мишень: кластер с 20+ сценариями уязвимостей: побег из контейнера, SSRF к metadata API, избыточные RBAC, отсутствие network policies.
