# 🔑 Эталонные решения

> **⚠️ СПОЙЛЕРЫ!** Эта папка содержит примеры решений для всех заданий курса.

---

## Как пользоваться

1. **Сначала сделайте задание сами.** Серьёзно. Смысл курса — в практике, не в копировании ответов.
2. **Застряли?** Загляните в решение конкретного задания — не всего этапа.
3. **Сделали?** Сравните своё решение с эталонным. Различия — это нормально. Если ваше решение обосновано — оно может быть лучше эталонного.

## Уровни подсказок

Каждое решение структурировано от подсказки к полному ответу:

- **💡 Hint** — направление мысли (без ответа)
- **📋 Approach** — пошаговый подход (без конкретных значений)
- **✅ Solution** — полное эталонное решение
