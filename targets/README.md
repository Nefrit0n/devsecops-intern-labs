# 🎯 Мишени (Vulnerable Apps)

Мы не изобретаем велосипед — используем лучшие open-source уязвимые приложения, каждое под свою задачу.

---

## Архитектура мишеней

```
                    ┌─────────────────────────────────────────┐
                    │         OWASP Juice Shop                │
                    │     🥇 основная мишень курса             │
                    │     Node.js · Angular · SQLite · REST   │
                    │                                          │
                    │  Этапы: 0, 1, 2, 3, 5                   │
                    └─────────────────────────────────────────┘
                                     │
            ┌────────────────────────┼────────────────────────┐
            ▼                        ▼                        ▼
  ┌──────────────────┐   ┌───────────────────┐   ┌────────────────────┐

  │  🔑 секреты       │   │  🔌 API security   │   │  ☸ K8s security     │
  │                  │   │                   │   │                    │
  │  Этап: 1         │   │  Этап: 3          │   │  Этап: 4           │
  └──────────────────┘   └───────────────────┘   └────────────────────┘
```

---

## Когда что поднимать

Не нужно ставить всё сразу. Каждый этап подскажет, какую мишень запустить.

| Мишень             | Когда нужна                   | Запуск                            |
|--------------------|-------------------------------|-----------------------------------|
| **Juice Shop**     | Этапы 0–3, 5 (основная)      | `docker run -p 3000:3000 bkimminich/juice-shop` |
| **WrongSecrets**   | Этап 1, лаба по секретам      | `docker run -p 8080:8080 jeroenwillemsen/wrongsecrets` |
| **crAPI**          | Этап 3, лаба API security     | `docker compose up` (свой compose) |
| **Kubernetes Goat**| Этап 4, лаба K8s security     | Требует K8s-кластер (minikube)    |
