# Глоссарий

| Термин | Определение |
|--------|-------------|
| **РБПО** | Разработка безопасного программного обеспечения |
| **SAST** | Static Application Security Testing — статический анализ кода |
| **DAST** | Dynamic Application Security Testing — динамический анализ |
| **SCA** | Software Composition Analysis — анализ сторонних компонентов |
| **SBOM** | Software Bill of Materials — спецификация состава ПО |
| **STRIDE** | Методика классификации угроз: Spoofing, Tampering, Repudiation, Info Disclosure, DoS, Elevation of Privilege |
| **DFD** | Data Flow Diagram — диаграмма потоков данных |
| **Trust boundary** | Граница доверия — точка, где уровень доверия к данным меняется |
| **КИИ** | Критическая информационная инфраструктура |
| **ФСТЭК** | Федеральная служба по техническому и экспортному контролю |
| **Quality gate** | Критерий прохождения этапа пайплайна (pass/fail) |
| **Finding** | Результат сканирования — потенциальная уязвимость |
| **False positive** | Ложное срабатывание сканера |
| **CVE** | Common Vulnerabilities and Exposures — идентификатор уязвимости |
