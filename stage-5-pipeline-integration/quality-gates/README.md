## Замыкаем цепочку

На этапе 0 вы создали **требования безопасности**. На этапах 1–4 настроили инструменты. Сейчас вы замыкаете цепочку:

```
Угроза (этап 0) → Требование → Инструмент → Finding → Quality Gate → Pass/Fail
```

Quality gate отвечает на вопрос: **«Можем ли мы релизить?»** Не абстрактно, а конкретно: «У нас 0 CRITICAL findings в SAST и SCA, ZAP baseline чистый, Dockle не нашёл root user → **да, релизим**».
