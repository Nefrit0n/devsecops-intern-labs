# docker-labs

Лабораторные работы по Docker и container security для Intern-уровня.

## Лабораторная 1: Сборка образа

**Задача:** собрать Docker-образ учебного приложения.

**Команды:**
```bash
docker build -t vulnerable-app:lab ./vulnerable-app
docker images | grep vulnerable-app
```

**Ожидаемый результат:** образ `vulnerable-app:lab` создан локально.

**Вопросы:**
- Что делает каждая ключевая инструкция в Dockerfile?
