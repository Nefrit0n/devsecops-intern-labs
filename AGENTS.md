# AGENTS.md

## Назначение
Этот репозиторий предназначен для учебной практики Intern-уровня по DevSecOps. Любые изменения должны сохранять простоту, воспроизводимость и учебную ценность.

## Правила для coding agent
1. Соблюдать простую и учебную структуру материалов.
2. Не использовать облачные сервисы и закрытые/платные продукты как обязательную часть выполнения.
3. Отдавать предпочтение локально запускаемым open-source инструментам.
4. Не усложнять архитектуру без необходимости.
5. Все примеры должны быть воспроизводимыми на обычном ноутбуке.
6. Все команды и пути должны работать в Linux/macOS (по возможности OS-neutral).
7. Каждая лабораторная должна содержать:
   - цель;
   - шаги выполнения;
   - ожидаемый результат;
   - вопросы на понимание.
8. Тон материалов: нейтральный, профессиональный, ориентированный на обучение.
